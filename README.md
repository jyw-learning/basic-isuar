## 🛠️ Built With

- ⚛️ **React** - For the magic
- ⚡ **Vite** - For the speed
- 🎨 **CSS** - For the beauty

## 🚀 Want to Run This?

```bash
# Clone it
git clone https://github.com/isuar/portfolio-suar.git

# Jump in
cd portfolio-suar

# Install the goodies
npm install

# Fire it up!
npm run dev
```

Then visit `http://localhost:5173` and enjoy! 🎉
